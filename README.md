# 🎮 Marfel Fontanilla - Game Dev Portfolio

A modern, dark-themed portfolio website for a game development student featuring Concept Art, Drawings, and Environment Art portfolios.

## ✨ Features

- 🎨 Modern dark theme with gradient accents
- 🖱️ Custom cursor animations
- 📱 Fully responsive (mobile, tablet, desktop)
- 🖼️ Filterable gallery with lightbox
- ⚡ Smooth scroll animations
- 🎯 Contact form (Formspree integration)
- 🔍 SEO-friendly
